# Portfolio Project Plan v1

## Project Overview

Build a premium personal portfolio website for Le Hoang Thien.

The portfolio should present the journey of a Product-Focused Engineer who is evolving toward Software Engineering through real-world product development, system thinking, and engineering growth.

The website is intended to support job applications, recruiter evaluation, and professional presentation.

---

# Project Goals

## Primary Goals

* Present engineering journey
* Showcase product impact
* Demonstrate software engineering mindset
* Support job applications
* Increase recruiter engagement

## Secondary Goals

* Create a professional online presence
* Organize project experience into case studies
* Provide a centralized professional profile

---

# Positioning

## Core Statement

Product-Focused Engineer Designing Reliable Systems at Scale

## Supporting Narrative

Building products, solving real-world problems, and continuously growing toward software architecture and system design.
