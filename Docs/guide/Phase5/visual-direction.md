# Phase 5 — Visual Direction

## Design Intent
- Tone: Premium, calm, confident.
- Personality: Engineering-first, không phô trương.
- Experience goal: Người xem nắm được năng lực trong 30-60 giây đầu.

## Visual Principles
1. Typography dẫn dắt nội dung.
2. Whitespace rộng để tăng độ rõ và nhịp đọc.
3. Color dùng tiết chế, tập trung tương phản chức năng.
4. Motion chỉ phục vụ hierarchy và orientation.

## Style References
- Apple: sạch, tập trung nội dung.
- Linear: sharp và hệ thống rõ.
- Vercel: technical nhưng tinh gọn.

## Content-first Rules
- Hero phải rõ positioning statement ngay dòng đầu.
- Tránh icon decoration không phục vụ thông tin.
- Tránh card noise và hiệu ứng gradient quá đậm.

## UI Tone by Section
- Home: ấn tượng nhưng điềm tĩnh.
- Projects: scan nhanh, trọng tâm case studies.
- Project detail: readability cao, ưu tiên đoạn narrative + metrics.
- Timeline: tuyến tính, tiến triển nghề nghiệp rõ.
- Contact: ngắn gọn, chuyển đổi nhanh.
