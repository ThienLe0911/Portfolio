# Phase 5 — UI/UX Specification (Plan)

## Objective
Chuyển IA đã khóa ở Phase 4 thành bộ specification sẵn sàng cho development, đảm bảo đúng định hướng premium, minimal, engineering-focused.

## Scope
- Visual direction
- Design system foundations (color, typography, spacing, radius, shadow)
- Motion specification
- Responsive rules (mobile/tablet/desktop)
- UI states và accessibility baseline

## Inputs
- Phase 4 IA docs trong thư mục Phase4
- Phase 2 content bundle
- Phase 3 case study drafts
- Discovery + Requirement specs

## Deliverables
1. `visual-direction.md`
2. `design-system.md`
3. `motion-spec.md`
4. `responsive-rules.md`
5. `task.md`
6. `testcase.md`
7. `phase5-handoff.md`

## Success Criteria
- Dev có thể bắt đầu dựng UI mà không cần làm lại cấu trúc.
- Theme light/dark có token rõ ràng và nhất quán.
- Motion đủ tinh tế, không gây nhiễu nội dung.
- Responsive rules rõ breakpoints và behavior từng section.
