# Phase 2 — Content Architecture (Plan)

Mục tiêu
- Soạn thảo và chuẩn hóa nội dung cho Landing Page (Hero, About, Metrics, Timeline, Learning, Contact).
- Kết quả đầu ra: bộ copy hoàn chỉnh (short + long), danh sách ảnh/asset cần thiết, và file nội dung sẵn sàng để dev tích hợp.

Deliverables
- Hero Content: Headline, Subheadline, CTA (desktop + mobile suggestions).
- About Content: short blurb (1–2 dòng), expanded story (3–5 đoạn), engineering journey bullets.
- Metrics Content: tối đa 4 metrics (label, description, nguồn).
- Career Timeline: danh sách milestones có ngày và link tới project case study.
- Learning Section: các chủ đề đang học + 1–2 câu mô tả mỗi chủ đề.
- Contact Section: social links, email, resume download link.

Phạm vi (Scope)
- In-scope: nội dung text, đề xuất ảnh/illustration, alt text, và metadata cần thiết cho dev.
- Out-of-scope (Phase 2): layout/visual design chi tiết, implementation frontend (sẽ làm ở Phase 6).

Success criteria (Tiêu chí chấp nhận)
- Thien duyệt và chấp nhận bản copy final.
- Mọi asset cần thiết được lưu ở các thư mục đã chỉ định.
- Nội dung có cấu trúc rõ ràng, có thể chuyển thẳng sang file Markdown/JSON cho dev.

Timeline đề xuất (estimate)
- Tổng: 5–7 ngày làm việc (có thể điều chỉnh theo feedback).
  - Ngày 1: Thu thập asset & phân tích tài liệu hiện có.
  - Ngày 2: Draft Hero + About (1st pass).
  - Ngày 3: Draft Metrics + Timeline.
  - Ngày 4: Draft Learning + Contact + tích hợp resume.
  - Ngày 5: Tích hợp assets, hoàn thiện bản draft để review.
  - Ngày 6–7: Review + chỉnh sửa theo feedback.

Nguồn dữ liệu & assets (đã có)
- Resume / CV: Docs/info
- Projects: Docs/myu & id-myu, Docs/events-myu-vn, Docs/Hung Hau House
- Master plan / Discovery / Requirement: Docs/guide/MasterPlan.md và các tài liệu Discovery/Requirement hiện có

Dependencies & rủi ro
- Nếu thiếu số liệu cụ thể cho Metrics, sẽ dùng placeholder và cần Thien cung cấp số thực tế.
- Thiếu ảnh/screenshots dự án sẽ làm chậm việc tích hợp visual.

Assumptions
- Tone: professional, product-focused, Apple-inspired (premium, minimal).
- Nội dung sẽ được cung cấp/duyệt ở mức copy; frontend sẽ handle layout & responsive.
